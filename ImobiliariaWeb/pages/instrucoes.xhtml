Para executar o software Sistema Imobiliária Baixe o Java Runtime Environment 7. JRE Download

Em Java Platform, Standard Edition, Click em JRE Download.

Click em Accept License Agreement, na lista de Download selecione a plataforma do seu sistema.

Click em Instalar.

Aguarde o Processo de Instalação Terminar.

Click em Fechar.

Após Baixar o Software Sistema Imobiliária, vá a Diretório do Arquivo, Click com o Botão Direito, posicione o Mouse no menu Abrir com: e selecione a Opção: Java(TM) Platform SE binary.
